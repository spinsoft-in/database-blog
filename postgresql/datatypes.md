###### Numeric Datatypes

- tinyint (1 byte) ( -128 to 128)
- smallint (2 byte )
- mediumint ( 3 byte )
- int ( 4 byte )  ( -2147483648 to 2147483647)
- bigint ( 8 byte)


###### Fixed Point Datatypes

- DECIMAL() - e.g DECIMAL(5,2) 5 is the precision and 2 is the scale. -999.99 to 999.99 ( Max no of decimals - 65)


##### Character Datatypes

- char(size)
- varchar(size)

###### Date/Time Datatypes

- date
- time
- datetime
- timestamp
- year

###### Other Datatypes
- text
